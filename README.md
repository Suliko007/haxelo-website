# Haxelo Website

This is the official website for Haxelo, an endemic in-game advertising platform that enables seamless ad integration matching your game's art style and enhancing player experience.

## Features

- **For Game Publishers**: Boost revenue with ads that feel like a natural part of your game world
- **For Advertisers**: Reach engaged gaming audiences with targeted endemic ads
- **Legal Pages**: Terms & Conditions and Privacy Policy

## Technology Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Lucide React for icons
- Radix UI components

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Project Structure

- `/app`: Next.js app router pages
- `/components`: Reusable UI components
- `/public`: Static assets
- `/styles`: Global styles and Tailwind config
